## Visão Geral

O EcoTrip Impact Calculator é uma aplicação web que permite aos usuários calcular o impacto ambiental (emissão de CO₂) de suas viagens em diferentes meios de transporte. O projeto objetiva conscientizar sobre a pegada de carbono e incentivar escolhas mais sustentáveis no planejamento de deslocamentos.

A ferramenta fornece estimativas de emissões para diversos modos de transporte, permitindo comparações e ajudando na tomada de decisões mais ecológicas.

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica da aplicação
- **CSS3**: Estilização responsiva e layout moderno
- **JavaScript (ES6+)**: Lógica de cálculo e interatividade
- **GitHub Copilot**: Assistência durante o desenvolvimento

## Processo de Criação

1. **Definição do Escopo**: Identificação dos meios de transporte e fórmulas de emissão de CO₂
2. **Estrutura HTML**: Criação do formulário de entrada com campos para distância e tipo de transporte
3. **Estilização CSS**: Design responsivo e interface intuitiva
4. **Implementação JavaScript**: Cálculos de emissões, validação de dados e dinâmica da interface
5. **Refinamento**: Otimização de código e melhorias na experiência do usuário

## ✅ Resultados

 Calculadora funcional e responsiva
- Visualização clara dos resultados de emissão em kg de CO₂
- Interface amigável que promove consciência ambiental
- Comparação entre diferentes meios de transporte
- Acessibilidade em dispositivos móveis e desktop

## Reflexão

Este projeto demonstra como a inteligência artificial (através do GitHub Copilot) pode acelerar o desenvolvimento web mantendo a qualidade do código. O desafio reforçou a importância de:

- **Sustentabilidade**: Desenvolver ferramentas que promovam mudanças de comportamento positivas
- **Simplicidade**: Criar interfaces que transmitam dados complexos de forma clara
- **Acessibilidade**: Garantir que a aplicação seja usável por diversos públicos
- **Colaboração Humano-IA**: Aproveitando as capacidades da IA para iteração mais rápida

O desenvolvimento com assistência de IA permitiu focar na lógica de negócio enquanto a ferramenta auxiliava em padrões de código repetitivos, resultando em uma solução mais eficiente e sustentável no tempo de desenvolvimento.
